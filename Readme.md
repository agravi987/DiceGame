## 🎮 Game Features

- Two-player dice game
- Timer countdown for each turn (10 seconds)
- Rolling a **6** gives an extra chance
- First player to score **100** wins
- Rolling dice animation
- Dark blue modern UI with glowing effects

## 🛠️ Tech Stack
